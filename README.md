# Dota 2 实时总资产悬浮插件 (Dota 2 NetWorth Overlay)

这是一个为 Dota 2 玩家设计的轻量级实时总资产（Net Worth）悬浮插件。它通过 Dota 2 官方提供的 **Game State Integration (GSI)** 接口获取实时游戏数据，并在屏幕上以简洁的悬浮窗形式展示当前英雄的总资产（包含金钱、所有装备价值、阿哈利姆魔晶及神杖加成）。

## 📖 开发背景

在 Dota 2 比赛中，玩家通常需要手动计算或查看计分板来估算自己的经济状况。为了让玩家在对局中能更直观地掌握自己的经济节点（如关键装备更新、买活钱预留等），本项目开发了一个实时更新的总资产悬浮窗。它不仅能提供数据支持，还能根据游戏状态自动显示或隐藏，确保不干扰玩家的正常操作。

## 📊 数据来源

1.  **Dota 2 GSI (Game State Integration)**: 核心数据源。插件在运行时会自动将所需的 GSI 配置文件写入 Dota 2 目录，从而接收来自游戏的 JSON 数据包，包含：
    *   `player`: 实时金钱（Gold）。
    *   `items`: 各个格子的物品名称。
    *   `hero`: 阿哈利姆魔晶（Shard）和神杖（Scepter）的加成状态。
2.  **`item_price.json`**: 内置物品价格数据库。插件会将 GSI 获取的物品名称映射到此 JSON 文件中，累加计算所有装备的总价值。

## 🛠️ 技术栈与开发流程

*   **UI 框架**: 使用 **C# WPF** 构建高性能、透明且支持点击穿透的 Windows 悬浮窗口。
*   **后端服务**: 使用 C# 内置网络库搭建本地 HTTP 服务器（端口 3000），用于接收 Dota 2 发送的 POST 数据。
*   **状态管理**:
    *   设计了 **GSI 数据缓存池**，有效解决了 GSI 增量更新（Delta Update）可能导致的数据大幅波动问题。
    *   使用多线程/异步操作分离 HTTP 服务器、UI 线程和 Dota 2 进程监控逻辑。
*   **交互逻辑**:
    *   集成全局快捷键支持（默认 `Ctrl+Alt+F10`）。
    *   实现了配置保存机制（`config.txt`），自动记录悬浮窗位置、锁定状态及快捷键设置。

## ✨ 功能特性

*   **实时总资产显示**: 自动计算并展示金钱 + 装备价值 + 魔晶/神杖 buff。
*   **智能自动显隐**:
    *   当 Dota 2 进程未运行或玩家不在对局（如在主菜单）时，插件自动隐藏。
    *   仅在比赛开始或试玩模式中显示。
*   **位置自定义**: 解锁状态下支持鼠标任意拖拽，锁定后支持点击穿透，不影响游戏内点击。
*   **全局快捷键**: 随时通过组合键切换锁定/解锁状态。
*   **免配置 GSI**: 运行程序后会自动检测并写入 GSI 配置，无需手动操作。
*   **系统托盘支持**: 支持托盘运行、右键菜单退出及快捷键修改。

关于本插件安全性与 VAC 防封号的郑重说明（必看）

很多水友担心使用这个“实时总资产计算”插件会导致 Dota 2 账号被 VAC（Valve Anti-Cheat）封禁。**结论写在前面：本插件 100% 安全，绝对不会导致封号。**

为了让大家放心使用，这里向大家科普一下本插件的工作原理：

**1. 使用的是 V社官方提供的合法接口（GSI）**
本插件完全基于 V社官方的 **游戏状态集成（Game State Integration, 简称 GSI）** 技术开发。GSI 是 V社专门为赛事直播推流、外部数据统计以及雷蛇/罗技等外设灯光联动而开发的官方功能。这也是为什么你需要手动添加 `-gamestateintegration` 启动项的原因，因为你是在向 V社官方申请开启这项合法服务。

**2. 绝不读取内存，绝不修改游戏文件**
市面上被 VAC 封号的作弊软件（如修改皮肤、透视等），其原理是强行读取或注入 `dota2.exe` 的内存，甚至篡改游戏底层文件。
本插件**没有任何侵入性操作**。它的原理是：Dota 2 游戏客户端主动把当前合法的公开数据（你的金钱、物品栏）打包成一个 JSON 文本，发送到你电脑的本地端口。本插件仅仅是一个“接收文本并做加法运算”的计算器，全程连 Dota 2 的游戏进程都不碰，VAC 根本没有任何理由判定其违规。

**3. 没有“开全图”等不公平优势**
V社在 GSI 接口中做了严格的安全限制。在正常的匹配对战中，GSI **只会发送你当前视野内本就能看到的数据**（比如你自己的存款和你身上的装备）。它绝对无法获取战争迷雾中敌方的经济和位置，因此不存在任何破坏游戏平衡的作弊行为。
